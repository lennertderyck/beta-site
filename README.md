New Media Design II 2018-2019
=============================

Jekyll voorbeeld
----------------

### Installatie kloon

```
PS> c
PS> git clone «repositorynaam» «mapnaam»
PS> c «mapnaam»
```

### Updaten van de Bundle

```
PS> c «mapnaam»
PS> bundle update
```

### Starten van Jekyll

```
PS> c «mapnaam»
PS> bundle exec jekyll serve
```
